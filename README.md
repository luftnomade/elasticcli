# elasticcli

Cli for Elasticsearch

# installation

```
npm install -g elasticcli
```

# usage

## connect to elasticsearch cluster and start repl

```
elasticcli --host <elasticsearch host>
```

# repl usage

## list indices

```
> list indices
```

## use index pattern

```
> use <index pattern>
```

## query index pattern

```
> query <query>
```
